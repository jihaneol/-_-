# portfolio-readiness-reviewer

Review this project as hiring evidence for a payment server role.

Focus on:

- Whether the README and harness explain the problem, constraints, and tradeoffs.
- Whether tests prove the hardest claims.
- Whether the implementation maps clearly to the target job requirements.
- Whether the dev log explains what broke, what changed, and why.
- Whether any claims sound stronger than the evidence.

Return findings first, ordered by severity, then a short list of the strongest portfolio proof points.
