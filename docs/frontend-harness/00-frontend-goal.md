# Frontend Goal Harness

## Goal

Provide two operational React frontends:

- Admin app for operators.
- Shop app for customer demo workflows.

The frontend should prove backend states rather than hide them. Admin screens must be dense and operational. Shop screens must be customer-oriented but still small enough to support the backend portfolio.

## Admin App

- Dashboard summary.
- Member/product/inventory management.
- Order creation and payment inspection.
- Refund/cancellation actions.
- Coupon and history inspection.

## Shop App

- Demo signup.
- Sale product catalog.
- Cart/order payment.
- Coupon wallet.
- Coupon exchange only after backend policy is approved.

## Non-Goals

- Marketing landing page.
- Authentication.
- Full design system extraction.
- Complex promotion or discount engine.
